Build Number: 0.2

Date Released: 17/01/2020

Updates:

  • Error messages have been added to the sign in / sign up processes.
  • Added game updates / bug fixes information to the lobby screen.
  • The sign in process now requires an account to proceed.
  • Added a floating information pop-up that will show when hovering over each skill.
  • Began work on a skilling menu that will tell the player what they unlock at each level.
  • Devised an exp curve for gathering skills.
  • Players can now submit bug reports
  • The player's skill levels and experience are now loaded in from the database upon sign in
  • The player's inventory are now loaded in from the database upon sign in

Bug Fixes:

  • The player character will no longer move if the player's cursor is hovering over UI elements.
  • The camera will no longer zoom in if the player's cursor is hovering over UI elements.
